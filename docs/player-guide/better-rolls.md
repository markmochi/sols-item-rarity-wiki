# Find better rolls

There is no hidden luck trick: the best way to improve a roll is to obtain another fresh copy and compare the result.

## What “better” means

For weapons and armor, compare:

1. the underlying item itself;
2. rarity tier;
3. the stat package shown in the tooltip;
4. how well those stats suit your build.

For utility tools, also compare:

1. perk count;
2. perk names;
3. each Roman-numeral perk tier.

A Rare tool with the perk you need can be more useful than an Epic tool with a less relevant set. A Supreme tool is special because every default perk it receives is tier V.

## A practical hunt

1. Make or collect several copies of the same item.
2. Reveal each one by selecting it in the main hand.
3. Compare rarity first.
4. On tools, compare perk count and tiers next.
5. Keep the roll that best fits the job.
6. Store, trade, recycle, or use the others according to your modpack's normal systems.

Item Rarity itself does not add recycling or salvage.

## Know the pool ceiling

An omitted tier cannot roll. For example:

- wooden gear stops at Rare by default;
- stone and leather gear stop at Epic;
- normal iron gear stops at Legendary;
- diamond gear can reach Mythical but not Supreme;
- netherite gear can reach Supreme;
- tridents and turtle helmets can reach Supreme.

Check [Default roll tables](../reference/default-rolls.md) or use the [item lookup](../reference/item-lookup.md) before grinding for a tier that is not in the item's pool.

## Things that do not improve odds

- Luck armor or the Luck attribute
- Looting
- Fortune
- better enchantments
- killing stronger mobs
- opening rarer chests
- higher difficulty
- revealing in a different dimension
- letting another player reveal it

Those mechanics may help you obtain more items in ordinary Minecraft or another mod, but the rarity roll itself uses only the configured pool.

## Can an existing item change?

Normal pool edits do not change an already stored result. Dropping, trading, relogging, repairing, or hovering also does not reroll it.

A server's fixed-rarity override can change how an item displays because fixed rules outrank stored rolls. That is an admin rule change, not a player reroll.

!!! tip "Fastest route to more attempts"
    Focus on obtaining more copies of the exact item you want. More attempts—not a player stat—give you more chances to see the pool's high tiers.
