## Report a bug

Use the [guided bug report form](https://github.com/markmochi/sols-item-rarity-wiki/issues/new?template=bug_report.yml) for problems with the mod. Search the [issue tracker](https://github.com/markmochi/sols-item-rarity-wiki/issues) first so the same problem is not reported twice.

Before submitting, have these ready:

- Minecraft version
- Sol's Item Rarity version
- Forge or NeoForge version
- Single-player or dedicated-server environment
- Exact steps that reproduce the problem
- Other installed mods and their versions
- `latest.log`, `debug.log`, or the crash report
- Relevant config files, screenshots, or video

Remove passwords, private server addresses, access tokens, and other personal information before uploading files. The [bug-report guide](https://markmochi.github.io/sols-item-rarity-wiki/reference/reporting-issues/) explains where to find logs and how to make a report easy to reproduce.
